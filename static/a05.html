1. Beschreibung des Datenmodells

In dieser Aufgabe wurde eine kleine Chat-Datenbank umgesetzt. Das Datenmodell besteht aus drei Tabellen:

  • rooms – enthält Chat-Räume (id, name)
  • users – Beispielbenutzer (id, name)
  • messages – Nachrichten (id, room_id, user_id, author, text, created_at)

Jeder Raum kann beliebig viele Nachrichten enthalten, und jede Nachricht gehört genau zu einem Benutzer.

4. Wichtige SQL-Abfragen & DB-Funktionen

Der Datenbankzugriff ist gekapselt in src/sqlite.ts. Beispiele wichtiger Funktionen:

getAllRooms()
getAllUsers()
getMessagesForRoom(roomId)

Beispiele verwendeter SQL-Abfragen:

SELECT * FROM rooms;
SELECT * FROM users;
SELECT * FROM messages WHERE room_id = ? ORDER BY created_at ASC;
